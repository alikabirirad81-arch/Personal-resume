This is a **personal resume website** built to showcase my professional experience, projects, skills, courses, education, languages, and contact information.

## Features

- **Responsive design**: Mobile-first layout for better experience on small screens.
- **Dark/Light mode toggle**: Users can switch between dark and light themes.
- **Interactive navigation**: Smooth scroll and active section highlighting.
- **Project showcase**: Display projects with live links to Behance.
- **Animated skill bars**: Skill levels animate when the section comes into view.
- **Contact form**: Simple frontend contact form with placeholder alert for demo purposes.

## Technologies Used

- **HTML5** for structure.
- **CSS3 & custom properties** for styling and theme management.
- **JavaScript** for interactions and animations.
- **Google Fonts (Poppins)** for typography.

## Design Decisions

- **Sidebar navigation** for easy access to all sections.
- **Cards and sections** to visually separate experiences, projects, and skills.
- **Gradient accents** to make important elements stand out.
- **Accessibility considerations**: Keyboard navigation for sidebar, aria labels for screen readers.
